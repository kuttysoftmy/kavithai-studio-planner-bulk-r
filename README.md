# ⚡ Kavithai Studio Planner & Bulk Renderer

> Built autonomously by **Angela AutoPilot** based on [quotes-image-generator](https://github.com/kuttysoftmy/quotes-image-generator).

A companion workflow automation tool and marketing planner designed for Tamil content creators. Instead of designing quotes one by one, users can input a batch of quotes (or import a CSV), apply custom typography and color presets, visually review them on a multi-device social media grid preview, and bulk-download the rendered canvas images in a single ZIP file. It complements the original editor by scaling it into a high-throughput production tool with template automation and a content scheduling calendar.

## Tech Stack
React + Tailwind CSS + Vite + Canvas API + JSZip

## Quick Start
```bash
npm install
npm build
```
